# waiter_app
 Waiter App – Flutter (Clean Architecture)

A fully offline, clean-architecture structured Waiter Ordering App built using Flutter, following provider state management, Hive local database, and modular design.

This app allows waiters to browse menu items, manage categories, add items to a cart, place orders, and view complete order history — all offline.

 Features
 Menu & Categories

Loads menu and categories from local JSON

Displays items based on selected category

Default category selection

Menu item images included

 Cart System

Add / remove items

Adjust quantity with animations

Swipe to delete items

Cart total displayed in bottom bar

 Order Management

Place orders & save them locally (Hive DB)

Clear cart after placing order

Order history screen with:

Summary

Time & date

Item count

Total amount

 Order Detail Screen

Shows full order details

Shows previous orders section

Smooth transitions

 Offline First

Everything works without internet

Using Hive for fast local storage

Local JSON as menu source
