# waiter_app
 Waiter App – Flutter Assignment

A clean, modular, offline-first Waiter application built using Flutter, Provider, Hive, and Clean Architecture principles.

This project demonstrates:
    - Clean Architecture layered design
    - Local JSON parsing
    - Category-wise menu filtering
    - Cart → Order → History flow
    - Local persistence using Hive
    - Figma-like UI components
    - Smooth animations 
    - Reusable and scalable structure
